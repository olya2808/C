Вводится строка из строчных латинских букв, слова разделены пробелами, признак конца
ввода - точка. При выполнении задания предложение организовать в виде двухсвязного списка слов.
Программа должна содержать функции формирования исходного списка, вывода списка до и
после модификации и реализации непосредственно варианта задания.

Перед минимальным словом (в лексикографическом смысле) вставить элемент с его инверсией.
